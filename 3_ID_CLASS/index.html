ID e CLASS

ID

o ID fimciona como um CPF. Assim como só posso ter um numero de CPF, cada elemento só pode ter uym valor de ID. O CPF é único, ou seja, ninguém pode ter um CPF igual ao meu. isso significa que o ID também é único, então não é possível repetir o mesmo valor duas vezes.

CLASS

o CLASS fumciona de maneira oposta ao ID. Diferente do ID em que você só pode ter um valor, no CLASS voçê pode ter vários valores. também diferente do ID, em que não se pode repetir um valor, no CLASS você pode ultilizar o mesmo valor para vários elementos.
